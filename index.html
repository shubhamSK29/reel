How It Works

1

Encrypt

Your password is encrypted with AES-GCM using your master password

2

Split

The encrypted data is split into multiple shares using Shamir's Secret Sharing

3

Hide

Each share is embedded into a different image using steganography

4

Recover

Use any threshold number of images to reconstruct and decrypt your password

Security Features

  • No Single Point of Failure: Password is distributed across multiple images
  • Threshold Security: Only need a subset of images to recover (e.g., 2 out of 3)
  • Steganographic Concealment: Hidden data is invisible to casual inspection
  • Layered Encryption: Multiple cryptographic primitives for enhanced security
  • Offline Operation: No reliance on online services or cloud storage
