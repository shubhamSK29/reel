Troubleshooting Tips:

  • Verify that you're using the correct master password
  • Ensure the binary file or base64 data is valid and uncorrupted
  • Check that the data format matches what was originally encrypted
  • For base64 data, make sure there are no extra spaces or line breaks

About Manual Decryption

Manual decryption is used for legacy compatibility or when you have encrypted data that was saved as binary files or base64 strings instead of being embedded in images.

Supported Formats:
  • Binary Files (.bin): Raw encrypted data saved to disk
  • Base64 Strings: Text representation of encrypted binary data
